# Kadon's Personal Portfolio

A modern, clean, and responsive one-page portfolio website built with Bootstrap 5.

## Features
- **Modern Design**: High-end aesthetics with glassmorphism and smooth gradients.
- **Night Mode**: Built-in dark/light mode toggle with persistence.
- **Responsive**: Fully optimized for mobile and desktop.
- **Micro-animations**: Interactive elements and scroll animations.

## Tech Stack
- HTML5 / CSS3
- Javascript (Vanilla)
- Bootstrap 5
- AOS (Animate On Scroll)
- Google Fonts (Inter)
